Download

You can download a minimized version of the library or get the full source code at github.com.

Download source

Install with npm

$ npm install coreui-panel

Требования

Для корректной работы на странице необходимо наличие: Bootstrap 5, jquery

Insert components

                            {id: "tab5", title: "Tab title 4", active: false, url: "data/tab3.json"},
                            {type: "divider"},
                            {id: "tab6",title: "Tab title 5", active: false, url: "data/tab4.json"}
                        ]
                    }
                ],
                content: "Raw denim you probably haven't heard of them jean shorts Austin. Nesciunt tofu stumptown aliqua, retro synth master cleanse. Mustache cliche tempor, williamsburg carles vegan helvetica. Reprehenderit butcher retro keffiyeh dreamcatcher synth. Cosby sweater eu banh mi, qui irure terry richardson ex squid. Aliquip placeat salvia cillum iphone. Seitan aliquip quis cardigan american apparel, butcher voluptate nisi qui."
            });

            panelAjax.render('example-ajax-content');


            panelAjax.on('click-tab.coreui.panel', function (tab, event) {
                console.log(tab.id + ' ' + 'click-tab.coreui.panel')
            });
            panelAjax.on('start-load-content.coreui.panel', function (tab, xhr) {
                console.log(tab.id + ' ' + 'start-load-content.coreui.panel')
            });
            panelAjax.on('success-load-content.coreui.panel', function (tab, result) {
                console.log(tab.id + ' ' + 'success-load-content.coreui.panel')
            });
            panelAjax.on('error-load-content.coreui.panel', function (tab, xhr, textStatus, errorThrown ) {
                console.log(tab.id + ' ' + 'error-load-content.coreui.panel')
            });
            panelAjax.on('end-load-content.coreui.panel', function (tab, xhr) {
                console.log(tab.id + ' ' + 'end-load-content.coreui.panel')
            });
        </script>

Options

Name type default description
id string crc32b Идентификатор панели. Позволяет получать панель для работы с ней, либо для css настроек. Если не указан, то будет равен случайному значению
title string Заголовок панели
subtitle string Подзаголовок панели
backUrl string Адрес для кнопки перехода "назад". Кнопка назад существует только если есть заголовок
controls string Любой Html код добавляющий свою функциональность
tabsType string tabs Вид табов. Напрямую зависит от bootstrap. Доступные варианты: tabs, pills, underline
tabsPosition string top-left Позиционирование табов в пределах панели. Доступные варианты: top-left, top-center, top-right, left, left-sideways, right, right-sideways
tabsFill string Выравнивание табов. fill, justify
tabsWidth int 200 Ширина колонки с табами. Может применяться совместно с позиционированием left, left-sideways, right, right-sideways
tabs array Массив объектов описанием табов для панели
content string | array | object Содержимое панели. Может быть в виде строки с html, в виде объекта coreui, либо массива с комбинацией этих объектов

Methods

let panel = CoreUI.panel.create( { ... } )

Метод для создания экземпляра панели.

let panel = CoreUI.panel.get('panel-id')

Получение ранее созданной панели по id

panel.render('element-id')

Формирует и вставляет сгенерированную html-панель в указанный элемент. Параметр элемента может содержать идентификатор элемента страницы или объект элемента страницы. Если параметр не указан, метод вернет сформированный html код панели

panel.initEvents()

Инициализировать события панели. При использовании метода panel.render('element-id') c переданным элементом, метод panel.initEvents() выполняется автоматически.

panel.getId()

Получение идентификатора панели

panel.setContent(content)

Установка содержимого внутри панели. Может содержать в себе тоже, что и параметр content в опциях

panel.on('click-tab.coreui.panel', function() { ... })

Установка события, которое может произойти в текущей панели.

Events

Event Type Description
shown.coreui.panel Показ панели на странице. Происходит один раз при отображении панели
show-content.coreui.panel Показ содержимого внутри панели. Первый раз происходит при показе панели, а после при каждом выполнении метода panel.setContent(content)
click-tab.coreui.panel Происходит при нажатии на таб.
В качестве параметров передаются panel.on('click-tab.coreui.panel', function (tab, event)
start-load-content.coreui.panel Происходит при клике на таб с указанным свойством url. Дает возможность показывать прелодер или модифицировать запрос через изменение объекта xhr.
В качестве параметров передаются panel.on('start-load.coreui.panel', function (tab, xhr)
success-load-content.coreui.panel Происходит при успешной ajax загрузке содержимого панели.
В качестве параметров передаются panel.on('success-load.coreui.panel', function (tab, result)
error-load-content.coreui.panel Происходит при НЕ успешной ajax загрузке содержимого панели.
В качестве параметров передаются panel.on('error-load.coreui.panel', function (tab, xhr, textStatus, errorThrown)
end-load-content.coreui.panel Происходит после завершения ajax запроса для получения содержимого для панели. Дает возможность скрыть прелодер.
В качестве параметров передаются panel.on('end-load.coreui.panel', function (tab, xhr, textStatus)
